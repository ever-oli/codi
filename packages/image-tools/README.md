# @mariozechner/pi-image-tools

Pi package that adds an `image_generate` tool backed by Google Antigravity image models.

Requires `/login` for the `google-antigravity` provider.

## Install

```bash
pi install /absolute/path/to/packages/image-tools
```

## Smoke Test

```text
Generate a simple 16:9 image of a lighthouse at dusk and save it to the project.
```
