# @mariozechner/pi-session-search-tools

Pi package that adds a `session_search` tool for finding relevant content in the current session or branch.

## Install

```bash
pi install /absolute/path/to/packages/session-search-tools
```

## Smoke Test

```text
Ask pi two or three questions first, then say:
"Use session_search for the phrase I mentioned earlier and show the matching session entry ids."
```
