# @mariozechner/pi-web-tools

Pi package that adds lightweight `quick_web_search` and `quick_web_extract` tools through Pi extensions.

## Install

```bash
pi install /absolute/path/to/packages/web-tools
```

## Smoke Test

```text
Use quick_web_search to find the official TypeScript handbook URL.
Use quick_web_extract on https://www.typescriptlang.org/docs/ and summarize the first section.
```
